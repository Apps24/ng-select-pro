## [0.1.0] - 2024-01-01

### Added
- Initial release of ng-select-pro
- Multi-select and single-select modes
- Search/filter functionality
- Virtual scrolling for large option lists (>100 items)
- Full keyboard navigation (ArrowUp, ArrowDown, Enter, Escape, Tab)
- ControlValueAccessor support (ngModel & ReactiveFormsModule)
- Select all / deselect all
- Configurable badge display with overflow indicator
- Group headers support
- Disabled options support
- Tooltip support on options
- Loading state with spinner
- CSS custom properties for full theming
- Tailwind CSS integration support
- Angular Material integration support
- Dropdown position: auto, top, bottom
- Angular animations for smooth open/close
- ARIA attributes for accessibility
